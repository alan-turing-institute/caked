A dataloader abstract package

## Installation

```bash
python -m venv env
source env/bin/activate
python -m pip install --upgrade pip
python -m pip install caked
```

From source:

```bash
git clone https://github.com/alan-turing-institute/CAKED
cd CAKED
python -m pip install .
```
